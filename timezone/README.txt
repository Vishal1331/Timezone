INTRODUCTION
------------

This module is responsible for Timezone features.


INSTALLATION
------------

 * Install module from route Extend and install module timezone.
 * Admin config Url:- admin/config//timezone/timezone-form
 * Fill value for Country, City and Timezone field and submit the form.
 * Go to the “Block Layout”. Eg:- Admin Menu > structure > block layout
 * Go to your block region where you want to put it.
 * Click the "Place block" button and in the modal dialog click the 
    "Place block" button next to "Timezone Config Block".
 * Click on the Save block button.

CONFIGURATION
-------------

Configuration page: admin/config//timezone/timezone-form

Form will be displayed on above url.
